## Description

First application, which was created to lern Nest technology.
In this app is possible to manage products. Management have been 
written with using CRUD.
Every product have it's own id, title, description and the price.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
